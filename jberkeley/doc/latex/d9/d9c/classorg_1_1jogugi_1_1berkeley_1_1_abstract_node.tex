Clase abstracta que implementa la interfaz {\ttfamily \doxyref{I\+Node}{p.}{d4/df5/interfaceorg_1_1jogugi_1_1berkeley_1_1_i_node}}. 

La clase {\ttfamily \doxyref{Abstract\+Node}{p.}{d9/d9c/classorg_1_1jogugi_1_1berkeley_1_1_abstract_node}} proporciona la implementación base para un nodo en el sistema. Maneja el contexto de Zero\+MQ, los sockets de comunicación y contiene las operaciones comunes para el envío y recepción de mensajes, además de almacenar la información del nodo como su nombre, dirección y tiempos de espera.
